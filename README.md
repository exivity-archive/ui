A collection of components to build interfaces.

Quickstart
----------

Install with `yarn`

```bash
yarn add @exivity/ui
```

Start using it

```jsx
import { Button } from '@exivity/ui'

<Button>Click me!</Button>
```

Visit [exivity.github.io/ui](https://exivity.github.io/ui/) for more documentation and examples.
