# Flex

A general purpose component to make flexbox layouts.

## Usage

```jsx
<Flex alignItems='center'>
  <Flex.Item paddingRight={1}>
    <Text>Watch out for this...</Text>
  </Flex.Item>
  <Flex.Item>
    <Heading>Bumaye!</Heading>
  </Flex.Item>
</Flex>
```

## Props

Both `Flex` and `Flex.Item` inherit all props from `Block`.
