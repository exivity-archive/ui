Installation
============

With **yarn**:

```bash
yarn add @exivity/ui [--ignore-optional]
```

With **npm**:

```bash
npm i @exivity/ui [--no-optional]
```

Dependencies
------------

### Peer dependencies

The following packages must be already installed in your project:

| dependency | version |
|------------|---------|
| react | ^16.8 |
| react-dom | ^16.8 |
